# Flashcards

Produce 10–14 flashcards as a JSON array. Each card:

```
{ "question": "...", "answer": "...", "tag": "...", "difficulty": "easy|medium|hard" }
```

Mix of:
- Recall (definitions, parts).
- Mechanism (what causes what, in one direction).
- Reverse mechanism (given an effect, name the cause).
- Misconception correction (state a wrong claim, ask why it is wrong).

The first 6 cards must be hard enough that a learner who only memorised
labels would fail them.
